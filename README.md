# pygen
Commandline asci generative art

Небольшая, пока ещё примитивная среда для создания генеративной asci-графики непосредственно в консоли.

Создано для использования в непосретственно на проекторе с Android TV с подключенной клавиатурой (лучше с тачпадом) через Termux. Для редактирования кода удобно использовать NeoVim. Для обмена файлами удобно использовать X-Plore и облачный диск (здесь-то и нужен тачпад).

См. превьюшку на проекторе: https://youtu.be/5PKT0Oir-EU


PS осторожно, очень плохой код =)
